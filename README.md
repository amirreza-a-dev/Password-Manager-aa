# Password Manager

A simple terminal-based password manager written in Python.

## Description

This project is a command-line password manager built to improve my Python skills while learning file handling, JSON, authentication, and encryption.

## Features

- [x] Add passwords
- [ ] View saved passwords
- [x] Search passwords

- [ ] Master Password authentication

- [ ] Encrypted password vault
